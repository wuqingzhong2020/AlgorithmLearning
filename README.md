Effective modern c++）和 c++ core guideline）， 第三方面则是据了解c++到c++ 2020版本后就想是另外一门语言了（习惯写法）， 个人认为未来长时间只要掌握到c++ 2017就够了， 后面的标准没兴趣学习， 然而rust可以替代后续的， 所以个人对rust的要求是掌握核心的工程和标准库，能够刷题就够，为未来储备一下技能。
